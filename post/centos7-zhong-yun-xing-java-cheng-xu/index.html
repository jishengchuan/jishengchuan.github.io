2.Centos7中运行java程序

2019-08-20

1.安装java运行环境

yum install java-1.8.0-openjdk-devel

2.在文件夹下如demo建文件

vim HelloWorld.java
输入a,i,o,r进入插入模式
public class HelloWorld{
	public static void main(String[] args){
		System.out.println("HelloWorld!");
	}
}
esc退出插入模式
:wq保存退出 :q!不保存退出 :q!退出

3.编译执行java代码

javac HelloWorld.java 
java HelloWorld
Hello World!
